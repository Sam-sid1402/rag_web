Machine Learning Projects

1. Real-Time Fraud Detection System

This machine learning project focuses on detecting fraudulent credit card transactions.

Main technologies:
- Python
- Scikit-learn
- XGBoost
- LightGBM
- Optuna
- SHAP
- FastAPI

Key features:
- Feature engineering
- Hyperparameter optimization
- Threshold tuning
- Explainable AI
- Real-time prediction API
- Deployment to cloud services

Final performance:
- ROC AUC approximately 0.997
- F1 Score approximately 0.86
